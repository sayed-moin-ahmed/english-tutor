# Cheat Sheets — compact reference cards

Quote or hand these over when teaching a rule. Keep them visual; the learner should be able to screenshot one and use it.

## Articles: a / an / the / none

```
Is the noun countable & singular?
        │
   NO ──┴── YES
   │         │
 plural/     Does the listener already know WHICH specific one?
 uncountable      │
   │         YES ─┴─ NO
   │          │      │
 usually    "the"  "a / an"
 no article
 (or "the"
  if specific)
```

- **a** before a consonant *sound* (a house, a university /yu/), **an** before a vowel *sound* (an apple, an hour).
- **the** = definite: one specific thing you both know (*the meeting tomorrow*, *the sun*).
- **a/an** = indefinite: introducing one, non-specific (*a book*, *an idea*).
- **no article** = general plurals (*cats are independent*) and uncountables (*water is wet*, *I gave feedback*).
- **First-mention rule:** introduce with *a/an*, refer back with *the*.

## English tenses (the core 6)

| Tense | Form | Use | Example |
|---|---|---|---|
| Present simple | do/does | habits, facts | "I drink coffee every morning." |
| Present continuous | am/is/are + -ing | happening now / temporary | "I'm cooking dinner now." |
| Past simple | did | finished, specific time | "I called her yesterday." |
| Present perfect | have/has + done | past→now, unspecified time | "I've finished the book." (done now) |
| Past continuous | was/were + -ing | ongoing past, interrupted | "I was sleeping when it rang." |
| Future (will / going to) | will + base / going to | predictions / plans | "I'll help." / "I'm going to travel." |

**Perfect vs past:** specific past time (*yesterday, at 3pm*) → past simple. "Ever/never/yet/already/since/for" or result-now → present perfect.

## Common prepositions (learn as chunks)

- **Time:** *at* 3pm, *on* Monday, *in* June / 2026 / the morning.
- **Place:** *at* the office, *in* the room, *on* the table.
- **Verb + prep collocations:** depend **on**, good **at**, interested **in**, responsible **for**, listen **to**, arrive **at** (a place) / **in** (a city), apologize **for**, consist **of**.
- Tip: memorize the verb+preposition as one unit, never the preposition alone.

## Capitalization

- First word of a sentence; the pronoun **I**; proper nouns (names, places, brands: *London*, *Sarah*, *Toyota*); days/months (*Monday, June*) but **not** seasons (*summer*); titles before names (*Dr. Lee*).
- Not: common nouns (*the city*, *a teacher*) unless they start a sentence.

## Commas (the frequent four)

1. Between items in a list: *a, b, and c*.
2. Before a coordinating conjunction joining two full sentences: *I called him, but he didn't answer.*
3. After an introductory phrase: *After the rain, the sky cleared.*
4. Around a non-essential aside: *The book, which I read twice, was great.*
- **Avoid the comma splice:** don't join two full sentences with only a comma (*It rained, we stayed home* → use *so*, or a period, or a semicolon).
